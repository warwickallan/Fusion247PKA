**Disposition: Surface for Warwick.** Not Promote, not Enrich, not Discard, not bare Retain-source-only.

Filing "here is how to build a raw/wiki/schema personal knowledge system with an agent-governed ingest/maintain/lint loop" into `ai-tooling.md` as a new discovery would be restating, from a different creator's presentation, an architecture Warwick is already the daily operator of — this is the "actually redundant" case, not merely "conceptually similar." That distinguishes it cleanly from the Hermes pilot, whose content (a specific competing product's feature set and marketing claims) genuinely was new information about something Warwick didn't already have.

**Why not flat Discard or silent Retain-source-only:** four *tactical, sub-architecture* patterns in the video are genuinely not yet part of myPKA and are the kind of implementation choice only Warwick (not Cairn) should greenlight, because they touch the team's actual build, not a personal AI-tooling fact:

2. **An explicit vault-write firewall / allow-list** restricting an agent to one vault (Chunk 12) — a security-boundary pattern arguably adjacent to Vex's territory once App-Developer-Pack-style concerns apply here, not currently modeled in this scaffold.
3. **Scheduled/heartbeat automation** for the ingest→maintain→lint loop (mentioned in Chunk 10's wrap, "once a week or once a day... it will automatically run this entire loop") — currently a manual, on-demand trigger in myPKA (session-log triggers, on-demand SOP-017), not scheduled.
4. **Web-Clipper-style raw capture** (Chunks 2, 9) — a lighter-weight general-purpose analog to the deferred TubeAIR adapter, but for arbitrary web pages, not just YouTube.

These are genuine, if narrow, architecture-evolution candidates — exactly what GL-010 means by "high-impact, preference-sensitive" implications that should go to Warwick rather than be quietly enriched into a Topic note (wrong shape of knowledge for `ai-tooling.md`, which is Warwick's personal tool-usage notes, not a systems-design backlog) or silently dropped (they're not nothing).
