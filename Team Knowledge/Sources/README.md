# Sources

Home for **source-derived knowledge notes** — standalone, human-readable notes distilled from
external sources (starting with YouTube transcripts via TubeAIR → Cairn). This folder is the
governed vault location the BUILD-002 Unified Fusion Hub files into.

- **Governed + git-tracked** general knowledge (this is the PUBLIC `Fusion247PKA` repo). No personal
  or entrusted data is ever filed here — personal material stays in `PKM/` / the private domain
  stores, never on the public repo (personal-data doctrine).
- Each note is a **standalone knowledge reconstruction**, not a transcript copy or a thin summary:
  metadata + provenance, executive orientation, substantive knowledge, claims vs. verified facts,
  Fusion247 interpretation separated from source content, justified backlinks, and an honest review
  state (`ai_created` / `pending_warwick_review` / `approved`).
- The **immutable RAW transcript** is preserved separately as evidence and linked from the note.
- Obsidian opens `Team Knowledge/` as the governed vault (WP1). The Obsidian Local REST API plugin
  (localhost-only, authenticated) is the write path for the hub worker; its key lives in the Yoga
  secret store, never in git.
